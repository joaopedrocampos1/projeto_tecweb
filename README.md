# 🌐 Projeto de Desenvolvimento Web

**Tecnologias Web — Prática Presencial (Unidades 1 e 2)**

## 🎯 Objetivo Geral

Este projeto tem como objetivo o desenvolvimento de uma aplicação web utilizando **HTML, CSS e JavaScript**, abordando conceitos fundamentais do desenvolvimento frontend. A proposta inclui a implementação de navegação entre páginas, criação de formulários com validação, listagem dinâmica de dados e gerenciamento de estado em memória, simulando um cenário real de desenvolvimento.


## 📂 Entrega do Projeto

O projeto contem:

* 📁 Repositório hospedado no GitHub
* 📌 Organização das tarefas utilizando **GitHub Projects (Kanban)**
* 🧾 Commits organizados e coerentes com a evolução do desenvolvimento
* ▶️ Aplicação funcional executável localmente


## 🚀 Como Executar o Projeto

1. Clone o repositório:

   ```bash
   git clone <link-do-repositorio>
   ```

2. Acesse a pasta do projeto:

   ```bash
   cd nome-do-projeto
   ```

3. Abra o arquivo `index.html` em seu navegador.
